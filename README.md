# Caviar y Trufa

Sitio web de Caviar y Trufa — caviar y trufa de origen, Cancún, México.

Sitio estático (HTML + CSS + JS vanilla), sin dependencias ni build.

## Desarrollo local

```bash
python3 -m http.server 4173
# abrir http://localhost:4173
```

## Publicación

Desplegado con GitHub Pages desde la rama `main`.
